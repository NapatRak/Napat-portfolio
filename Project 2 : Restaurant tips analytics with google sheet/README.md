โปรเจคนี้ผมจะทำการวิเคราะห์พฤติกรรมการให้ทิปและปัจจัยที่มีผลต่อยอดใช้จ่ายในร้านอาหาร โดยใช้เทคนิคทางสถิติและการทำ 
Data Visualization เพื่อหาความสัมพันธ์ระหว่างยอดบิล, ช่วงเวลา (วัน/มื้ออาหาร), และลักษณะของลูกค้า (ขนาดกลุ่ม/การสูบบุหรี่) 
เพื่อสร้าง Insight ในการบริหารจัดการบริการและประมาณการรายได้จากทิปอย่างแม่นยำ
